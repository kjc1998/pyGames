# pyGames

## Prerequisites

- `python` >= 3.12
- `pip` >= 25.0
- `make`

## Installation

1. Create and activate a [virtual environment](https://docs.python.org/3/library/venv.html)

   ```bash
   > python -m venv .venv
   > source .venv/bin/activate
   ```

2. Install `pygames` package

   ```bash
   > make install # standard
   # or
   > make devinstall # develoepr
   ```

3. Run tests

   ```bash
   > make tests
   ```
